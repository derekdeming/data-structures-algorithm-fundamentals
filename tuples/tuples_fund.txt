a tuple is an immutable sequence of python objects 

tuples are also comparable and hashable 

t = 'a', 'b', 'c', 'd', 'e'
t = ('a', 'b', 'c', 'd', 'e')


creating a tuple: time complexity -- O(1), space complexity -- O(n)
traversing a given tuple: time complexity -- O(n), space complexity -- O(1)
accessing a given element: time complexity -- O(1), space complexity -- O(1)
searching a given element: time complexity -- O(n), space complexity -- O(1)
